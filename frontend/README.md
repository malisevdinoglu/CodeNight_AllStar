# Frontend

GECICI STUB: Osman'in React 18 + Vite + TypeScript uygulamasi gelene kadar
port 3000'i ayakta tutan minimal nginx container'i.

Osman icin not: Bu klasoru kendi yapinla degistirebilirsin; tek sozlesme,
container'in 80 portundan servis vermesi (compose `3000:80`) ve tum API
cagrilarinin `http://localhost:8080/api/v1/...` uzerinden yapilmasi.
